
#  JimWhere – 창고/보관함 예약 & 결제 서비스

JimWhere는 사용자가 원하는 창고/보관함을 기간제로 예약하고
결제 → 출입 관리 → 이용 내역 확인까지 한 번에 처리할 수 있는
실사용형 B2C 공간 대여 플랫폼입니다.
